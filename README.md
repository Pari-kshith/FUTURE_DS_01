# FUTURE_DS_01 – Business Sales Performance Analytics

## 📌 Project Overview

This project focuses on analyzing online retail sales data to identify revenue trends, top-selling products, and country-wise sales performance. The analysis was performed using Python in Jupyter Notebook with the help of data visualization libraries.

---

## 🎯 Objective

The main objective of this project is to:

* Analyze business sales data
* Identify monthly revenue trends
* Find top-selling products
* Analyze country-wise sales performance
* Generate meaningful business insights

---

## 🛠️ Tools & Technologies Used

* Python
* Jupyter Notebook
* Pandas
* Matplotlib
* Seaborn
* VS Code

---

## 📂 Dataset

Dataset Used: `online_retail.csv`

The dataset contains:

* Invoice details
* Product descriptions
* Quantity sold
* Unit price
* Customer details
* Country information
* Invoice date

---

## 📊 Analysis Performed

The following analyses were performed:

* Data Cleaning
* Revenue Calculation
* Monthly Revenue Trend Analysis
* Top Selling Products Analysis
* Country-wise Sales Analysis
* Data Visualization using Charts and Graphs

---

## 📈 Visualizations

The project includes:

* Monthly Sales Trend Graph
* Top Products Revenue Chart
* Country-wise Sales Performance Chart

---

## 🔍 Key Insights

* United Kingdom generated the highest sales revenue.
* A few products contributed significantly to overall sales.
* Sales increased during certain months showing higher customer demand.
* Some countries had very low sales compared to others.

---

## ✅ Conclusion

The project helped in understanding customer purchasing behavior and overall business performance using data analysis techniques. The analysis provided useful insights that can help businesses improve sales strategies and decision-making.
